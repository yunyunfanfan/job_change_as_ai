数据分析作业说明

大模型对IT行业的 xxx 影响分析

上面 xxx = “就业岗位市场” or “职业技能” or “就业岗位市场和职业技能”

收集数据、处理数据是 Python的核心应用场景，
本任务就此围绕我们关心的问题做一个训练。
基本情况，
2022年11月30日，OpenAI 发布基于大语言模型 GPT-3.5的ChatGPT。2023年初国内各个企业跟进发布自己产品。
如此这三年，技术不断进步，对社会影响不断展现，大家的认识不断改变

关于本问题
最直接的是就业数据。最近国外几个热点讨论都是基于招聘网站数据
但是，我发现国内主要几个招聘网站都只能看到当前数据，没法做对比
所以，一个替代数据建议：
选择某自媒体平台 ( 知乎、小红书、B站、v2ex... 国外的知乎、贴吧之类 )
选取这三年几个不同时间段的若干讨论主贴，获取主贴以及下面评论内容

数据基本要求：
如果是自媒体数据，要求不少于18个讨论主贴 (可以是跨平台的)，每个选中的主贴下面跟帖数据不少于100条
主贴的选取要人工辨识，是否引起了大家对岗位市场和技能的讨论。单纯关键词搜索不准确。也就是讨论虽然是关于大模型冲击下的 IT 技能和岗位，但是它多半不以这样的关键词直接出现。

最近这些年网络文本分析的文章，论文非常多。可以选择性的参考
数据分析的基础是以得到的数据 , 我们并不判定自媒体讨论内容对不对。
如果发现了其他有效的能回答题目问题的数据源,也可以不局限于自媒体或者不用自媒体数据

【 考核点 】
数据质量
分析报告
代码要求能支持复现报告里面的一些工作，不做工程性质量要求


【作业提交】
一个压缩文件 “安卓_数据分析” (如果小组名为安卓)，里面包括
1）报告，pdf格式，命名“安卓_报告.pdf”, 篇幅 8-25页 。图文并茂，以获取的数据为依据。

     对 data 做解释：如何得到数据，数据规模，分析技术方案和思路 。

3）附件.zip: 数据和源代码（ jupyter 或者 py 文件）
